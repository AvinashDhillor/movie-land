# Movie Land

This is a simple movie app that interacts with [OMDb API](http://www.omdbapi.com/) to get information about movies.

## Installation

```bash
$ git clone git@github.com:matrixersp/movie-land.git
$ cd movie-land && npm install
$ npm start
```
